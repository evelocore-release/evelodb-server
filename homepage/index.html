SDK
Simple by design

Spend less time wrestling infrastructure. The EveloDB SDK is expressive, typed, and stays out of your way.

query.ts evelodb-global 
import eveloDB from 'evelodb-global'

const db = new eveloDB({
    host: '127.0.0.1',
    port: 7962,
    user: '...',
    key:  '...'
})

async function find() {
    const res = await db.find(
        'users',
        { age: { $gte: 18 } }
    )
    const users = res.data.all()

    console.log("Response:", res)
    console.log("Users:", users)
}

Features
Engineered for excellence

Everything needed to ship data-intensive applications without the operational overhead.

⚡
Sub-Millisecond Latency

Optimized memory mapping delivers query responses under 1ms, even under sustained throughput.

💎
Native BSON Support

Store complex nested structures with binary efficiency and full type safety — no schema required.

🛡️
ACID Compliant

Strict transactional integrity ensures data is always consistent, isolated, and durable.

🖥️
Built-in Admin Panel

A fully-featured management UI ships out of the box. Monitor, query, and manage visually.

🔗
TCP + HTTP Transport

Raw TCP for high-throughput real-time workloads or HTTP for standard REST-style access.

📦
Zero Configuration

Running in seconds. No YAML sprawl, no provisioning scripts, no maintenance windows.
